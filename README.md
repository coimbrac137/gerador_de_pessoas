# 🧪 Gerador de Dados Sintéticos Brasileiros

## Sobre o Projeto

Este projeto é uma ferramenta CLI desenvolvida em **Node.js** que gera dados demográficos e documentais fictícios no padrão brasileiro. Foi criado para auxiliar desenvolvedores, testadores e profissionais de dados que precisam de informações realistas sem violar a LGPD (Lei Geral de Proteção de Dados).

## 🎯 Objetivos

### 1. Segurança e Privacidade
- Eliminar a necessidade de usar dados reais em ambientes de teste
- Estar em conformidade com boas práticas de proteção de dados
- Fornecer uma alternativa ética ao uso de informações pessoais

### 2. Produtividade
- Acelerar o desenvolvimento com dados prontos e variados
- Reduzir o tempo gasto criando dados de teste manualmente
- Permitir a geração em massa para estresse de sistemas

### 3. Realismo
- CPFs que seguem o algoritmo oficial de validação
- Nomes e endereços culturalmente apropriados ao Brasil
- Distribuição realista de idades e locais

## 👥 Público-alvo

- **Desenvolvedores** que precisam popular bancos de dados de desenvolvimento
- **QAs e Testers** que automatizam testes de cadastro
- **Professores** que ensinam programação e bancos de dados
- **Designers** que criam protótipos com dados reais
